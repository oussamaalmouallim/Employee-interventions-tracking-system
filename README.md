# Employee Interventions Tracking System

## Overview
The Employee Interventions Tracking System is designed to manage and track various interventions made in response to employee needs. This project aims to streamline the communication and documentation of interventions within organizations, improving employee satisfaction and operational efficiency.

## Technologies Used
- **Frontend:**
  - HTML, CSS, JavaScript
- **Deployment:**
  - Docker for containerization
  - Github pages for hosting

## Features
- Dashboard to view employee interventions
- Search and filter options for interventions
- Real-time notifications for updates

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/oussamaalmouallim/Employee-interventions-tracking-system.git
   ```
2. Navigate to the project folder and install dependencies:
   ```bash
   cd Employee-interventions-tracking-system
   npm install
   ```
3. Start the application:
   ```bash
   npm start
   ```
